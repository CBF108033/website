健體端勢
想要健身卻沒有專人從旁指導? 健體端勢就是您的最佳好助手! 他可以幫助你一起訓練並且即時給予姿勢糾正，讓您能夠正確無憂地運動，加上我們為您建立的歷史記錄和分析資料，隨時都能夠掌握自己的運動成果。

功能 Function
    • 運動姿勢模型
    • 健體端勢內建最基本的三大動作模型，分別是深蹲、橋臀、弓步蹲，透過多組角度作為正確姿勢的判斷。
    • 即時修正回饋
    • 健體端勢擁有文字和語音提示，當動作不正確時能夠即時給予回饋，降低運動姿勢錯誤時帶來的代償作用。
    • 訓練成果
    • 在主介面的右下角會即時顯示動作次數和熱量消耗，讓健身者運動時更能夠專注在運動上。
特色 Feature
    • 體感介面
    • 透過特定動作和觸發點能夠進行動作切換、暫停/繼續運動等多項功能，讓健身者無須走到電腦前就可操控系統。
    • 登入系統
    • 健體端勢設有登入功能，即使有多位使用者使用系統，資料都不會混亂。我們還設有google登入方便使用者轉移設備，且資料也能安全地備份在雲端。
    • 歷史紀錄
    • 健體端勢建立了歷史紀錄，不管多久以前的資料都能夠被完整保存，也能夠點擊各筆資料看更多詳細資料。
    • 分析報告
    • 健體端勢設有三種模式的分析報告，分別是年報表、月報表、周報表，透過多種模式觀察運動成果，客製化運動清單。
技術 Technology
  • Python
  • 健體端勢系統以Python作為開發語言，如人體姿勢估計的MediaPipe和系統GUI-Tkinter。
  • OpenCV + MediaPip
  • OpenCV廣泛用於圖像分析、圖像處理、檢測、識別等。
    MediaPipe是一款由Google開發的免費開源跨平台機器學習管線框架， 可以用來處理影像、聲音等時間序列資訊。其內建快速的機器學習，即便是在一般效能的硬體上也能有效運作。
  • Machine Learning
  • 健體端勢的模型以卷積神經網路（CNN）作為分類器，卷積神經網路結構主要是由卷積層、池化層、扁平層與全連接層組成，通過卷積/池化的方式把大量引數降維成少量引數再做處理，因此在處理影像資料上更有效率。

Step 1

蒐集資料

▼
資料來源分兩大類，主要於多個開放式圖片庫蒐集大量深蹲、橋臀和弓步蹲的動作圖片，次要是團隊成員們自行拍攝的影像作為部分訓練及測試資料。
Step 2

整理、分類動作

▼
為求更精準地預測動作，我們移除雜訊過多的圖片避免機器學習之成效不佳，例如圖片中同時出現兩人以上、背景較複雜、運動器材遮蔽過多人體部位之圖片。在分類的部分我們透過程式自動將圖片分到三動作資料夾中，降低手動分類大量圖片時所耗費的時間。
Step 3

CNN模型訓練

▼
CNN卷積神經網路相對DNN傳統神經網路保有更高的空間資訊及參數量較少之優勢。我們以三層卷積層和池化層作為整個神經網路架構，以提升模型精確度。最後將訓練完成的權重保存下來，以利後續應用時能夠直接使用。
Step 4

即時畫面動作訓練

▼
將權重資訊匯入本系統中，等待鏡頭開啟後可以直接偵測使用者的動作，並即時將結果呈現在系統介面上。
